# Shaheera Imtiaz — Portfolio

Personal portfolio built with **Nuxt.js 3 + Vue.js**, blending two custom design systems: **Doodle** (playful, hand-drawn) and **Pacman** (retro 8-bit arcade).

## Tech Stack

- **Framework:** Nuxt.js 3 + Vue.js 3
- **Styling:** Pure CSS — Doodle × Pacman design system
- **Fonts:** Delius Swash Caps · Press Start 2P · JetBrains Mono
- **Email:** Resend API
- **Containerization:** Docker + Docker Compose

## Sections

- Hero — animated avatar, tech chips, stats
- About — bio, info grid, downloadable CV
- Skills — categorized tags, proficiency bars
- Experience — timeline with impact metrics
- Projects — SAM, SkillSwap, Video AI Pro, EduConnect
- Education & Certifications
- Contact — form with email delivery via Resend

## Getting Started

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Environment Variables

```bash
cp .env.example .env
# Add your Resend API key
RESEND_API_KEY=re_xxxxxxxxxxxx
```

## Docker

```bash
docker compose up --build
```
